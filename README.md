# 🚀 Hackathon Event Registration System

A full-stack **Hackathon Event Registration System** built to manage participant registrations efficiently.  
This project uses a simple and responsive frontend with a powerful **Spring Boot backend** and an **SQL database** to store and manage data securely.

---

## 📌 Features

- 📝 Hackathon registration form
- 🎨 Responsive and clean UI
- ⚙️ RESTful backend using Spring Boot
- 🗄️ SQL database integration
- ✅ Server-side validation
- 🔐 Secure data handling
- 📱 Works on all screen sizes

---

## 🛠️ Tech Stack

### Frontend
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)

### Backend
- Java Spring Boot  
- REST API  
- Maven

### Database
- SQL (MySQL / PostgreSQL / H2)
